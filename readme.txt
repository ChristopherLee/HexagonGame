1. QUICK START

Following steps explains how to compile and run the player program.
After running xbuild shell, two folders - 'build' and 'dist' will be created 
under the current directory.
 
a: run shell script xbuild to compile the source codes to a jar:
./xbuild 

b: run shell script xrun to run a game with a human player and a random number of AI players:
./xrun 

You can run our unit tests by calling ./xtest
